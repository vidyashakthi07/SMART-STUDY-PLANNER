# Smart Study Planner

Smart Study Planner is a simple and effective web application designed to help students plan, manage, and track their daily study tasks. The application provides a clean interface to organize subjects, set goals, and maintain productivity.

## Features

- User Authentication (Signup/Login) using Firebase
- Add, edit, and delete study tasks
- Organize subjects and study schedules
- Track daily learning goals
- Responsive and simple user interface
- Real-time data storage using Firebase
- Accessible from any device via browser

## Technologies Used

- HTML
- CSS
- JavaScript
- Firebase Authentication
- Firebase Firestore Database
- Render (for deployment)
- GitHub (version control)

## How to Run Locally

1. Download or clone the repository
2. Open project folder
3. Open index.html in browser

OR

Use Live Server in VS Code

## Firebase Setup

1. Create project in Firebase Console
2. Enable Authentication (Email/Password)
3. Enable Firestore Database
4. Copy Firebase config into firebase.js

## Deployment

This project is deployed using Render as a Static Site.

Steps:
1. Push code to GitHub
2. Connect GitHub repo to Render
3. Select Static Site option
4. Deploy project
